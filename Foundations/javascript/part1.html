Variables

A variable is a "named storage" for data, essentially we define a name to store some sort of information. Within Javascript we use the let keyword to define a variable.

We can define a variable first and then assign it data, or we can combine the two: let variable; variable = 'test'; or let variable = 'test';

We can also define multiple variables in-line: let user = 'user', let john = 'John', let age = 25; however it is best practice to separate them on different lines.

Variable naming

When naming variables in Javascript there is a naming convention for variables called camelCase, where instead of having an underscore for whitespaces, we separate them using Capital Letters. Along this, the name must only contain letters, digits or the symbols '$' and '_', and the first character must not be a digit. Some examples: newVariable, variable1234, _testVariable1234, wowThisVariableIsReallyLong.

Constants

Another way to define variables in JS (short for Javascript) is the keyword 'const', like so: const variable = 'variable'; However constants CANNOT have their values changes.
